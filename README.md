codeforces-commons
==================

Commons for Codeforces and related projects.

Add repository reference to Maven:

~~~~~
<repositories>
   <repository>
       <id>codeforces-commons</id>
       <url>https://github.com/Codeforces/codeforces-commons/raw/master/repo/maven/</url>
   </repository>
</repositories>
~~~~~
